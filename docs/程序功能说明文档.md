# 工程项目管理系统 - 程序功能说明文档

## 1. 系统概述

### 1.1 系统简介
工程项目管理系统是一个基于Web的工程项目信息管理平台，提供项目信息的录入、查询、统计、导入导出等功能。系统集成OCR智能识别技术，支持从工程申报表单图片中自动提取项目信息，大幅提升数据录入效率。

## 2. 仪表盘模块

### 2.1 功能概述
仪表盘是系统的首页，提供项目数据的可视化概览，包括统计卡片、图表展示和快捷入口。

### 2.2 统计卡片

#### 2.2.1 项目总数卡片
- 显示当前系统中的项目总数
- 点击可跳转到项目列表页面

#### 2.2.2 许可证数卡片
- 显示状态为"许可证"的项目数量
- 点击可跳转到筛选后的项目列表

#### 2.2.3 待考评项目卡片
- 显示状态为"待考评"的项目数量
- 点击可跳转到待考评列表页面

#### 2.2.4 月度统计卡片
- 右上角下拉按钮选择年份（从2025年开始）
- 显示新增项目数：基于"标化工地申报时间"字段月度统计
- 显示考评项目数：基于"市标化检查时间"字段月度统计
- 支持月度/季度数据切换

### 2.3 图表展示

#### 2.3.1 状态分布图
- 饼图展示三种状态的项目数量分布
- 状态类型：正常、待考评、许可证
- 显示各状态数量及总数

#### 2.3.2 区域分布图
- 堆积柱形图展示各区域项目分布
- 支持图表/表格视图切换
- 显示全部13个区域数据
- 表格视图包含区域名称和数量

### 2.4 最近项目列表
- 显示最近添加的5个项目
- 包含项目名称、区域、状态、申报时间
- 点击项目名称可查看详情

## 3. 项目管理模块

### 3.1 项目列表

#### 3.1.1 功能说明
项目列表页面展示所有工程项目信息，支持搜索、筛选、分页功能。

#### 3.1.2 操作说明

| 操作 | 说明 |
|------|------|
| 搜索 | 在搜索框输入关键词，支持项目名称、申报企业、项目经理模糊搜索 |
| 区域筛选 | 下拉选择区域，筛选指定区域的项目 |
| 状态筛选 | 下拉选择状态，筛选指定状态的项目 |
| 分页 | 支持每页显示10/20/50/100条记录 |
| 添加项目 | 点击"添加项目"按钮进入添加页面 |
| 查看详情 | 点击项目名称进入查看页面 |
| 编辑项目 | 点击操作列的编辑按钮进入编辑页面 |
| 删除项目 | 点击操作列的删除按钮（需密码验证） |

#### 3.1.3 列表字段

| 字段 | 说明 |
|------|------|
| 申报工程名称 | 项目全称，点击可查看详情 |
| 申报企业 | 施工单位名称 |
| 项目经理 | 项目负责人姓名 |
| 区域 | 项目所在区域 |
| 状态 | 当前状态（正常/待考评/已考评/许可证） |
| 施工许可证取得日期 | 用于列表排序 |
| 操作 | 编辑、删除按钮 |

### 3.2 添加项目

#### 3.2.1 功能说明
添加项目页面用于录入新的工程项目信息，支持手动填写和OCR智能识别两种方式。

#### 3.2.2 表单字段

| 字段名 | 是否必填 | 说明 |
|--------|----------|------|
| 申报工程名称 | 是 | 项目全称 |
| 工程造价（万元） | 否 | 数字，保留4位小数 |
| 申报企业名称 | 否 | 施工单位全称 |
| 项目经理姓名 | 否 | 项目负责人姓名 |
| 联系人及电话 | 否 | 联系方式 |
| 监理企业名称 | 否 | 监理单位全称 |
| 项目总监姓名 | 否 | 监理方总监姓名 |
| 参选证号码 | 是 | 格式：市政+四位数字，全局唯一 |
| 参建企业名称 | 否 | 参建单位全称 |
| 创建目标 | 否 | 下拉选择 |
| 区域 | 否 | 下拉选择（13个区域） |
| 建设单位 | 否 | 建设方全称 |
| 开工日期 | 否 | 日期选择器 |
| 竣工日期 | 否 | 日期选择器 |
| 标化工地申报时间 | 否 | 日期选择器 |
| 施工许可证取得日期 | 否 | 日期选择器 |
| 市标化检查时间 | 否 | 日期选择器 |
| 检查人员 | 否 | 多人用顿号分隔 |
| 备注 | 否 | 文本域 |
| 状态 | 是 | 下拉选择，默认"正常" |
| 预打分 | 否 | 数字，0-100 |

#### 3.2.3 OCR智能识别功能

##### 功能说明
系统集成基于RapidOCR的智能识别功能，可以从工程申报表单图片中自动提取以下信息：

| 识别字段 | 提取规则 | 准确率 |
|----------|----------|--------|
| 工程名称 | 识别"工程名称"标签后的内容 | 100% |
| 工程造价 | 提取数字金额（万元） | 100% |
| 施工单位 | 识别"施工单位"相关标签 | 90%+ |
| 项目经理 | 识别"项目经理"后的姓名 | 100% |
| 联系电话 | 提取11位手机号 | 100% |
| 监理单位 | 识别"监理单位"后的公司名 | 100% |
| 项目总监 | 识别"项目总监"后的姓名 | 100% |
| 建设单位 | 识别"建设单位"后的公司名 | 100% |
| 开工日期 | 解析日期格式 | 100% |
| 竣工日期 | 解析日期格式 | 100% |

##### 操作步骤
1. 点击"OCR识别"按钮
2. 选择工程申报表单图片（支持jpg、jpeg、png格式）
3. 系统自动识别并填充表单
4. 检查并修正识别结果
5. 点击"保存"完成添加

##### 注意事项
- 图片大小限制：16MB
- 支持格式：jpg、jpeg、png
- 识别时间：约1.5-2秒
- 建议使用清晰、完整的表单图片

#### 3.2.4 参选证号码输入优化

##### 功能说明
参选证号码输入框实现了简化输入机制：
- 用户只需输入四位数字
- 系统自动添加"市政"前缀
- 仅接受数字输入
- 限制输入长度为四位
- 实时验证唯一性

##### 输入示例
- 用户输入：`0441`
- 系统存储：`市政0441`

#### 3.2.5 按钮布局

| 模式 | 顶部按钮 |
|------|----------|
| 添加模式 | 保存 + 返回列表 |
| 编辑模式 | 保存 + 取消 + 返回列表 |
| 查看模式 | 编辑 + 返回列表 |

### 3.3 编辑项目

#### 3.3.1 功能说明
编辑项目页面用于修改已有项目信息，表单字段与添加页面相同。

#### 3.3.2 操作说明
1. 在项目列表点击"编辑"按钮
2. 修改需要更新的字段
3. 点击"保存"完成编辑

#### 3.3.3 注意事项
- 参选证号码修改后需保证唯一性
- 系统自动记录更新时间

### 3.4 查看项目

#### 3.4.1 功能说明
查看项目页面以只读方式展示项目详细信息。

#### 3.4.2 操作说明
- 点击项目名称进入查看页面
- 页面展示所有字段信息
- 点击"编辑"按钮可进入编辑模式

### 3.5 删除项目

#### 3.5.1 功能说明
删除项目功能用于移除不需要的项目记录。

#### 3.5.2 安全验证
删除操作属于敏感操作，需要验证管理密码：
1. 点击"删除"按钮
2. 弹出密码验证对话框
3. 输入正确的管理密码
4. 确认删除

#### 3.5.3 注意事项
- 删除操作不可恢复
- 建议删除前先备份数据库

## 5. 系统设置模块

### 5.1 功能概述
系统设置模块提供数据导入导出、数据库备份恢复、自动备份配置等功能。

### 5.2 数据导入导出

#### 5.2.1 数据导出

##### 功能说明
将项目数据导出为CSV文件，支持按时间范围筛选。

##### 操作步骤
1. 选择"标化工地申报时间范围"
2. 点击"导出CSV"按钮
3. 浏览器自动下载CSV文件

##### 导出字段
所有项目字段，包括：
- 基本信息：项目名称、工程造价、申报企业等
- 日期信息：开工日期、竣工日期、申报时间等
- 状态信息：状态、预打分等

#### 5.2.2 数据导入

##### 功能说明
从CSV文件批量导入项目数据。

##### 操作步骤
1. 点击"选择文件"按钮
2. 选择CSV文件
3. 点击"导入"按钮
4. 系统显示导入结果

##### 注意事项
- CSV文件需使用UTF-8编码
- 参选证号码不能重复
- 导入会跳过已存在的参选证号码

### 5.3 数据库管理

#### 5.3.1 手动备份

##### 功能说明
手动创建数据库备份文件。

##### 操作步骤
1. 点击"立即备份"按钮
2. 系统自动创建备份文件
3. 备份文件存储在`data/backups/`目录
4. 文件命名格式：`backup_YYYYMMDDHHMMSS.db`

##### 备份文件列表
- 显示最近5个备份文件
- 包含文件名、创建时间、文件大小
- 支持下载备份文件

#### 5.3.2 自动备份

##### 功能说明
配置定时自动备份数据库。

##### 配置选项

| 选项 | 说明 |
|------|------|
| 启用开关 | 开启/关闭自动备份功能 |
| 备份周期 | 按日/按周/按月 |
| 备份时间 | 指定备份执行时间 |
| 保留数量 | 保留的备份文件数量 |

##### 操作步骤
1. 开启自动备份开关
2. 选择备份周期（日/周/月）
3. 设置备份时间
4. 点击"保存配置"

#### 5.3.3 数据库恢复

##### 功能说明
从备份文件恢复数据库。

##### 安全验证
恢复操作属于敏感操作，需要验证管理密码：
1. 选择备份文件或上传新文件
2. 点击"恢复"按钮
3. 输入管理密码验证
4. 确认恢复操作

##### 注意事项
- 恢复操作会覆盖当前数据库
- 建议恢复前先手动备份
- 恢复后需重启应用

### 5.4 系统信息

#### 5.4.1 显示内容

## 6. 安全机制

### 6.1 敏感操作验证

#### 6.1.1 需要验证的操作
- 删除项目
- 数据库恢复
- 清除敏感认证状态

#### 6.1.2 验证流程
1. 用户点击敏感操作按钮
2. 弹出密码验证对话框
3. 用户输入管理密码
4. 系统验证密码正确性
5. 验证通过后执行操作

#### 6.1.3 验证有效期
- 密码验证通过后有效期：1分钟
- 有效期内无需重复验证

### 6.2 数据验证

#### 6.2.1 前端验证
- HTML5表单验证
- JavaScript实时验证
- 参选证号码格式验证
- 参选证号码唯一性检查

#### 6.2.2 后端验证
- 数据类型检查
- 必填字段验证
- 唯一性约束验证
- 业务规则验证

### 6.3 密码存储
- 使用bcrypt哈希加密
- 密码不明文存储

## 8. API接口

### 8.1 OCR识别接口

#### 请求
```
POST /api/ocr
Content-Type: multipart/form-data

参数：
- file: 图片文件（jpg/jpeg/png）
```

#### 响应
```json
{
    "success": true,
    "data": {
        "project_name": "工程名称",
        "project_cost": "1000.0000",
        "declaring_company": "申报企业",
        "project_manager": "项目经理",
        "contact_phone": "联系电话",
        "supervision_company": "监理单位",
        "project_director": "项目总监",
        "construction_unit": "建设单位",
        "start_date": "2024-01-01",
        "end_date": "2025-01-01"
    }
}
```

### 8.2 参选证号码校验接口

#### 请求
```
GET /api/certificate/check?number=市政0001
```

#### 响应
```json
{
    "exists": false
}
```

### 8.3 密码验证接口

#### 请求
```
POST /api/verify-password
Content-Type: application/json

{
    "password": "管理密码"
}
```

#### 响应
```json
{
    "success": true
}
```

### 8.4 敏感操作认证状态接口

#### 请求
```
GET /api/check-sensitive-auth
```

#### 响应
```json
{
    "authenticated": true
}
```

## 10. 常见问题

### 10.1 OCR识别问题

| 问题 | 解决方案 |
|------|----------|
| 识别结果不准确 | 使用清晰、完整的表单图片 |
| 部分字段未识别 | 手动补充未识别字段 |
| 识别时间过长 | 缩小图片尺寸或分辨率 |

### 10.2 数据导入问题

| 问题 | 解决方案 |
|------|----------|
| 编码错误 | 确保CSV文件使用UTF-8编码 |
| 参选证号码重复 | 系统自动跳过重复记录 |
| 字段格式错误 | 检查日期、数字字段格式 |

### 10.3 备份恢复问题

| 问题 | 解决方案 |
|------|----------|
| 备份失败 | 检查backups目录权限 |
| 恢复后数据丢失 | 恢复前先备份当前数据 |
| 密码验证失败 | 确认管理密码正确 |
